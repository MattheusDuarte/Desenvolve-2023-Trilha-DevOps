- **vaiables.tf** define as variavéis da nossa aplicação

#Comandos
``` shell
#1. formatar terraform
terraform fmt

#2. iniciar o terraform
terraform init

#3. Planeja infra
terraform plan

#4. Aplica as configurações
terraform apply

```
